# Dyslexia Screener

A Finnish dyslexia screening tool built with React. Presents a series of reading
exercises and measures performance. No backend — everything runs in the browser.

## Exercises

- **Pseudoword detection** — decide whether a word is real or made up
- **Word search** — find target words in a passage within 2 minutes
- **Syllable assembly** — build a word by selecting syllable tiles in order
- **Minimal pair detection** — choose the correct word from a Finnish minimal pair (vowel/consonant length)

## Tech stack

Vite · React 18 · TypeScript · shadcn/ui · Tailwind CSS · React Router v6

## Getting started

```sh
npm install
npm run dev       # http://localhost:8080
```
